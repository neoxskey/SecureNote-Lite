## üîπ Option 1: Install via Android APK (Recommended)

This method allows you to install SecureNote Lite directly on your Android device without using the Play Store.

### Steps

1. Open the official project website:
   https://neoxskey.github.io/SecureNote-Lite/

   2. Click **Download APK**  
You will be redirected to GitHub Releases.

3. Download the file:
   SecureNote.apk

4. On your Android device:
   
- Open **Settings**
- Go to **Security / Privacy**
- Enable **Install unknown apps**
- Allow installation from your browser or file manager

5. Open the downloaded APK file and tap **Install**

6. After installation is complete, open **SecureNote Lite** and start using it
