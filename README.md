# SecureNote-Lite
SecureNote Lite is a simple, offline-first note app focused on privacy. All notes are stored locally on your device with no login, no ads, and no tracking. Includes basic note management and a built-in security audit checklist.
SecureNote Lite is a lightweight Android application designed for users who want a simple and private way to take notes.

The app works fully offline and stores all data locally on your device. No accounts, no internet connection, and no external services are required.

Key features:
• Create, edit, and delete notes
• Offline-first local storage
• No login, no ads, no analytics
• Built-in security audit checklist
• Clean and minimal interface
• Lightweight APK size

Privacy & Security:
• No data collection
• No tracking or analytics
• No network requests
• Notes never leave your device

SecureNote Lite is suitable for personal use, learning projects, and users who value privacy and simplicity.
